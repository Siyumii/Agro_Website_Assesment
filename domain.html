Literature Survey

Floriculture in Sri Lanka started as an industry in 1970. It has grown substantially during the last few years to become one of Sri Lanka's major foreign exchange generating ventures. Anthurium and Orchid are the most popular tropical cut flowers which are being grown commercially for exports as well as for the local market. Read More......

Research Gap

There are no existing applications built to monitor Anthurium plantations. Also existing applications do not provide the comfort of monitoring a whole plantation because all the aspects such as disease detection, growth prediction and risk management considering external environmental factors like humidity, sunlight and temperature are not performed in one single research. Another point noted was that the possibility of using existing applications in a real plantation is weak.

Research Problem

Anthurium is mostly grown in mild climates inside greenhouse environments. But major problems that large scale Anthurium growers in Sri Lanka face when monitoring their plantations are, protecting the plants from various bacterial and fungal diseases, knowing the outbreak of a disease and predicting the growth of a plantation due to changes in environmental conditions such as sunlight, temperature and humidity. Therefore growers need an efficient method to monitor their plantations tirelessly.

Research Objectives

General Objectives

Developing a user friendly stand-alone application for diseases detection, growth prediction and risk management

Specific Objectives

Detection of diseases and providing with causes and treatments: When a spot appears, the user will be warned using an alarm. Then causes and treatments are also given through the alarm so that user does not have to contact agricultural experts to treat the plants properly.

Growth prediction: This function warns the owner about the effective climate changes that affects the growth of a plantation. So the owner can take measure to avoid the problem before it affects.

Risk management: The user can know the environmental conditions affecting a certain disease through this function. If those environmental conditions are available, he can know that a certain disease is appearing

Methodology

Agro is a stand-alone application developed in order to assist large scale Anthurium growers in Sri Lanka to monitor their plantations efficiently. As developers, our main aim was to design and implement the application using the latest technology which will help them to save their time, effort and revenue. Read More......

Technologies Used

Python 2.7

An open source programming language used for general purpose programming. Python is widely used in GUI based desktop applications like image processing and graphic design applications.

OpenCV 3.2.0

Necessary features to train neural networks were obtained using the library OpenCV. It has a Python interface and supports Windows, Linux, Mac OS, iOS and Android.

JetBrains PyCharm Community Edition 2014.1.4

PyCharm IDE provides code analysis, a graphical debugger, an integrated unit tester, integration with version control systems. PyCharm is cross-platform with Windows, macOS and Linux versions

Raspberry Pi 3

Raspberry Pi is a series of small single-board computers. Raspberry Pi 3 is approximately 80% faster than the Raspberry Pi 2 in parallelized tasks.

Sensors

DHT 11 sensor was used to get the humidity and the temperature readings and the light-dependent resistor (LDR) to get the sun light intensity.

Camera

A digital camera (Fuji film Fine Pix S8300 compact camera) is used for image acquisition. This compact camera has a high quality lens which provides zooming. It provides the captured image to the system for further analysis.
